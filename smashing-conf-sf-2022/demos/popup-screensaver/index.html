screensaver.js

Because your site needs a screensaver, right? Using the new native popup coming to the web platform, we can promote content into what we refer to as the "Top Layer".

It's an accessible layer that sits above the rest of your content. You can create the "Modal" you've always wanted! We needed to "hack" this in the past...

You get a bunch for free when using popup. For example, the "Light dismiss". Click anywhere outside a popup or use pointer-events: none on it to dismiss. No event handling tricks. It's as easy as: 

<div id="myPopup" popup="popup">Popup content</div>

And then showing it with:

const POPUP = document.querySelector('#myPopup');
    POPUP.showPopup();

In fact, the only JavaScript at play here is to handle the timing and showing for this "Screensaver" demo. You can open a popup without any JavaScript at all! And the screensaver would work minus the color changes as it's nested marquees 🤫 

<button togglepopup="myPopup">Show screensaver without JavaScript</button>

There are lots of use cases for popup and things that can go into the "Top layer". It's an exciting time for the web platform!

This is currently behind an "Experimental flag" in Chrome Canary (chrome://flags ➡️ "Experimental Web Platform features" 🚀). But, it is coming soon! We want to make sure it's great for you first though! And your feedback is super appreciated 🙏
